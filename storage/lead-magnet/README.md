# Lead magnet PDFs

Place PDF files for lead-magnet guides here. The app serves them only after email capture (sent as email attachment).

**5 Caribbean Sauces guide:** Copy your PDF as:

- **Filename:** `5-caribbean-sauces.pdf`
- **Full path:** `storage/lead-magnet/5-caribbean-sauces.pdf`

If the PDF is in Cursor workspace storage, copy it from that location into this folder.
